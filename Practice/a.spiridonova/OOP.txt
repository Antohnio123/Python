1) Общий класс Heroes
поля:
- Name (входит в конструктор)
- Speed = 1 (по умолчанию)
- Weapon = gun (по умолчанию)
- ArtifactsList (список собранных артифактов)

методы:
- __init__(self, name) - конструктор
- сollect (self, artifact) - в метод включена проверка, есть ли у собираемого артифакта доп функции (коэффициент 
скорости, смена оружия и т.п., если есть, то меняются соответствующие переменные экземпляра у героя)
- shoot (self) - в методе используется имеющееся у героя weapon
- run (self) - в методе учитывается имеющийся у героя коэффициент скорости (меняется при сборе соответствующего 
артифакта)

2) Дочерний класс FlyingHeroes
- fly(self)

Объекты (Hero1, Hero2 и т.п. могут быть дочерними по отношению к общему классу Heroes или к его потомку FlyingHeroes

3) Класс контроллер GameController
поля:
- HeroesList (список действующих героев)
- ArtifactsList (список артифактов доступных для сбора)
методы:
- createHero(name)
- killHero(name)
- createArtifact(name, speedCoeff, weapon) - создаем артифакт, указывая, какой коэффициент скорости он дает
(если дает) и на какое оружие он заменяет оружие героя (если у артифакта есть такая функция), можно указать
другие доп. функции, которые дает артифакт
- deleteArtifact

4) Класс Artifact 
поля:
- speedCoeff
- weapon

и объекты этого класса
